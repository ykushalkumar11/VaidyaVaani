# VaidyaVaani

VaidyaVaani is a full-stack web application that helps users understand medical prescriptions using AI. The application analyzes uploaded prescription images, extracts medicine details, explains prescriptions in simple language, and provides voice output to improve accessibility.

This project was developed to make prescription information easier to understand for patients who may find medical terminology difficult.

---

## Features

- Upload prescription images
- AI-powered prescription analysis
- Simplified explanation of medicines
- Medicine tracker
- Text-to-Speech support
- Multi-language interface
- Responsive web application

## Running the Project

Clone the repository

```bash
git clone https://github.com/yourusername/VaidyaVaani.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build the application

```bash
npm run build
```

Run production build

```bash
npm start
```
